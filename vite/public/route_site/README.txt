PEK indoor route video
======================

File: pek_videoroute_E.MP4  (fallback: pek_videoroute_E.mp4 on case-sensitive disks)

This file is NOT in Git (large file; avoids GitHub LFS quota).

Local dev
---------
Place a copy in this folder, or download from your own URL.

Render / CI
-----------
Set **PEK_VIDEO_URL** to an HTTPS URL for the file. Build runs `scripts/render-build.sh`
which downloads it before `npm run build`.
